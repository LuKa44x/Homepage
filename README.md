# Homepage

A personal portfolio website built with **HTML & CSS**, structured responsively using modern layout techniques (Flexbox & CSS Grid).  
This project serves as the foundation of my personal homepage/portfolio showcasing sections like header, projects grid, and a contact footer.

## 🌟 Overview

This repository contains the source code for a simple static portfolio-style site.  
It includes:

- A responsive **header** with profile picture and about section
- A **main** section listing project cards
- A detailed **footer** with contact info and social links
- Responsive design built with media queries and modern CSS layout

## 🎯 Features

✅ Responsive layout that adapts from desktop to mobile  
✅ Flexbox and CSS Grid used for structured content placement  
✅ Semantic HTML sections (`header`, `main`, `footer`)  
✅ Clickable social icons linking to personal profiles  
✅ Screenshot assets for project previews  
✅ Clean, minimal aesthetic for portfolio presentation

## 🚀 What’s Been Done

- Added anchor links around icon images for social profiles
- Included project screenshots inside `assets/`
- Enhanced homepage content to be more personal
- Built a 3×2 responsive grid for projects with adaptive breakpoints
- Styled contact section with clear info and responsive stack behavior

This site is mobile-first with breakpoints for:

- **Mobile** (single column stack)
- **Tablet** (two column grid)
- **Desktop** (three column projects grid)

- **HTML5**

- Modern layout: **Flexbox & CSS Grid**
